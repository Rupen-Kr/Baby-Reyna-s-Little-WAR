HTML Tags:
1. <header> - Defines the top section of the page.
2. <nav> - Contains navigation links.
3. <main> - Main content area.
4. <section> - Divides the page into logical sections.
5. <footer> - Bottom section with copyright info.
6. <img> - Displays images.
7. <ul> - Creates an unordered list.
8. <form> - For user input on the contact page.
9. <button> - Interactive button for actions.
10. <input> - Allows text input in forms.

CSS Properties:
1. font-family - Sets the font style.
2. font-size - Adjusts the font size.
3. color - Changes text color.
4. margin - Adds spacing around elements.
5. background-color - Sets the background color of elements.

JavaScript and Bootstrap:
JavaScript is used for an alert button that welcomes users. Bootstrap is used for styling components like the navigation bar, forms, and layout features.
